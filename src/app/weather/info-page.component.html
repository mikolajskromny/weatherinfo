Weather service

  • On the page „Weather” we can search for city from the list.

    • Button „Show actual weather” shows us weather for all added cities.

    • Button „Show actual forecast” shows us 5-day forecast for all added cities

    • Button „Delete all” deletes all cities

    • Button „Delete” near every city deletes only this city

  • On the page „Settings” you need to add your own API key.

    • To generate your API key visit this site: https://openweathermap.org/appid
