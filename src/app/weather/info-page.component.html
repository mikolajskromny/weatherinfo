Serwis Pogodowy

  • W zakładce „Pogoda” wyszukujemy miasta z listy po czym je dodajemy (klawisz ENTER lub przycisk „Dodaj”)

    • Przycisk „Pokaż aktualną pogodę” pokaże nam pogodę dla wszystkich dodanych miast

    • Przycisk „Pokaż aktualną prognozę” wyświetli prognozę pogody na 5-6 dni do przodu

    • Przycisk „Usuń wszystkie miasta” usunie z naszej listy wszystkie dotychczasowe miasta

    • Przycisk „Usuń” przy każdym mieście, usunie tylko to jedno wybrane miasto

  • W zakładce „Ustawienia” należy dodać prywatny klucz API

    • Aby zaktualizować klucz API wystarczy wpisać ponownie nowy klucz

    • Aby usunąć należy kliknąć usuń

    • Aby wygenerować swój klucz API proszę udać się pod adres: https://openweathermap.org/appid
